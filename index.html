Who Am I?

I'm a full-stack developer based in Brooklyn, NY. Instead of tutorial-clone portfolios, I ship real, working software: an AI blog-drafting tool that generates my own long-form writing, a music player with a hand-written native Android plugin, a browser-based audio-to-video converter with a full test suite, and a couple of Python/Kivy apps that talk to real APIs.

Every repo I keep active runs its own CI — linting, type-checking, and tests on every push, with several auto-deploying to GitHub Pages or building signed Android bundles. I care as much about the pipeline as the product.

My Projects

Real, deployed work — pulled straight from GitHub

Forge (PostPilot)

A one-tap forensic blog-post generator powered by Gemini — feed it notes and a stance, it drafts a long-form biographical article plus YouTube and Medium metadata. Runs entirely client-side as an installable PWA; the API key never leaves your browser.

React TypeScript Vite Tailwind Gemini API
Live Demo Code

Muzio AI — Smart Music Player

A local-first, dark-themed music player for web and Android that auto-tags messy MP3s (artist, album, year, genre) via Gemini. Ships with a hand-written native Java plugin that queries Android's MediaStore directly and handles the API 33 permission split.

JavaScript Capacitor Java (Android) Gemini API IndexedDB
Code

Audio to YouTube MP4

Turns an audio file and cover art into a YouTube-ready 720p video, entirely in the browser — no upload, no video editor. Canvas + Web Audio + MediaRecorder pipeline, with a real Vitest suite, strict TypeScript, and CI running lint/test/build on every PR.

React TypeScript Canvas API Vitest
Live Demo Code

TMDB Movie Recommender

A cross-platform Kivy app for browsing and searching TMDB's catalog, with infinite scroll, a persistent local watchlist, disk-cached posters, and an installable Android APK via Buildozer. Core logic is unit-tested with Pytest and runs in CI.

Python Kivy TMDB API Pytest
Code

Game Recommender

A Kivy browser for the RAWG catalog with 30+ curated shelves (genre, platform, retro consoles), a wishlist/played tracker with rollup stats, and a similar-games chain that falls back from suggestions to series to genre matches.

Python Kivy RAWG API
Code

Pong Game

A classic Pong build with real ball physics, a scaling AI opponent, and touch controls tuned for mobile. Installable as a PWA with offline support, sound effects, and a GitHub Actions workflow that deploys on every push to main.
